东莞地铁线路图（2024年最新）

东莞是广东省第 4 个开通运营轨道交通的城市，首条运营线路是东莞2号线（东莞火车站 - 虎门火车站），于2016年5月27日开通运营。另外还有在建的东莞1号线和2号线三期，其中东莞1号线有望在2024年底开通。
