食品生产许可证

什么情况下需要《食品生产许可证》？

根据《食品安全法》第三十五条，以下情况需办理：

  1. 从事食品生产加工活动：
    • 包括预包装食品、散装食品、特殊食品（如保健食品、婴幼儿配方食品）、食品添加剂等；
    • 例外： ✅ 销售食用农产品（如未加工的水果、蔬菜）； ✅ 仅对食品进行简单分装（如将大包装茶叶分装为小袋，未改变性状）。
  2. 委托加工：
    • 若你委托其他企业代工生产食品，被委托方（生产工厂）需持有许可证，委托方（品牌方）无需办理。

办理条件

1. 基础资质

  • 已取得《营业执照》，经营范围包含“食品生产”；
  • 生产场所符合环保、消防等要求。

2. 生产场所要求

  • 厂房面积、布局与生产品类匹配（如乳制品需专用车间）；
  • 具备消毒、通风、防虫鼠设施；
  • 与污染源（垃圾场、化工厂等）保持安全距离。

3. 设备与工艺

  • 配备与生产规模匹配的设备（如杀菌机、灌装机）；
  • 工艺流程合理，符合食品安全标准。

4. 人员要求

  • 至少1名专职食品安全管理人员；
  • 生产人员需持有健康证明。

5. 制度文件

  • 建立原料采购、生产过程控制、出厂检验等管理制度；
  • 制定食品安全风险防控预案。

办理流程

Step 1：准备材料

  • 营业执照复印件；
  • 法定代表人身份证；
  • 生产场所产权证明或租赁合同；
  • 生产设备清单、工艺流程图；
  • 食品安全管理制度文件；
  • 产品执行标准（如企业标准需备案）。

Step 2：提交申请

  • 线上：登录“国家市场监督管理总局食品生产许可审批系统”（网址：https://spxk.gsxt.gov.cn）填报信息并上传材料；
  • 线下：到生产场所所在地的 省级或市级市场监管局 提交纸质申请。

Step 3：现场核查

  • 监管部门受理后，20个工作日内派专家组进行现场核查；
  • 核查重点： ✅ 生产环境是否符合要求； ✅ 设备与工艺流程是否匹配； ✅ 制度是否有效执行。

Step 4：产品检验

  • 对首批生产的食品抽样送检，由第三方检测机构出具合格报告（费用约2000-5000元/批次）。

Step 5：审批发证

  • 核查和检验通过后，10个工作日内发放《食品生产许可证》（电子证与纸质证同步生效）。

有效期与费用

  • 有效期：5年（到期前6个月需申请延续）；
  • 办理费用：
    • 官方不收费（国家取消行政收费）；
    • 第三方费用： ✅ 厂房改造：视规模而定（小型车间约5-20万元）； ✅ 检测费：2000-5000元/批次； ✅ 代办服务费：1-5万元（协助准备材料、应对核查）。

注意事项

  1. 分类申请：
    • 按食品类别申请（如“糕点”“饮料”），不同类别需分别审核；
    • 若新增品类，需申请变更许可证。
  2. 标签合规：
    • 生产的预包装食品需标注 SC 编号（格式为“SC+14位数字”）。
  3. 委托加工备案：
    • 若委托生产，需向市场监管局提交委托加工协议备案。
  4. 合规风险：
    • 无证生产最高可处货值金额 10-20倍罚款，吊销执照；
    • 生产场所搬迁需重新申请许可证。

建议操作

  1. 先小规模试产：通过小批量生产验证设备和工艺，降低整改成本；
  2. 咨询专业机构：委托食品工程公司设计厂房布局，确保符合审查要求；
  3. 利用政策优惠：部分地区对小微企业有厂房改造补贴，可咨询当地市场监管局。

若仍有疑问，可拨打 12315 咨询属地市场监管局，或登录国家市场监管总局官网查询细则。
