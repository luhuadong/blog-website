准备工作

在广州注册公司需准备的 5 项资料：公司名称、注册资金、注册地址、经营范围、股东信息。

其中，比较麻烦的是注册地址，因为在广州注册公司需要商用性质地址，而不能是住宅。需要提供租赁合同、房产证复印件、场地使用证明等材料。

公司名称

注册公司需要先起名，并提交核准。格式为：

行政区划 + 字号 + 行业表述 + 组织形式

建议多备用3-5个名字，避免核名重复，增加通过率。避免与被吊销、注销营业执照的企业名称一致，通常是不会核准通过的。可以在核名网址上检索一下看看名称能不能通过。

“行业表述”有很多类型，以“科技”为例，有以下几种：

  • 科技（计算机、通信和其他电子设备制造业）
  • 科技（科技推广和应用服务业）
  • 科技（软件和信息技术服务业）
  • 科技（专业技术服务业）
  • 科技（研究和试验发展）

需要综合考虑企业的核心业务、未来发展方向以及相关政策支持等因素来选择。

注册资金

7月1号以后开始实行实缴制度了，需要根据自己的实力填写（最低3万），注册资金不宜太高，太高所承担的责任就越多；也不能填太少会显得公司没实力。注册资金的多少，一定程度上表示这个公司的资本是否雄厚，有没有做大项目的能力，客户有时候会以注册资金作为一种合作上的考虑，大公司更容易被信任。

股东信息

法人、自然人股东身份证复印件（正反面：清晰），有几个股东、各自以什么出资、占多少股份、分别负责什么工作、如何表决等等，协商好并写到合伙协议里，分配股权不能太随意，某些股东所起的作用与其持股比例不匹配，容易引起很多意见，影响公司的发展。股份协议，其主要的作用是约定各个股东之间明确各自要做的事情，以及需要承担的责任，最后的盈亏如何分配等，可以很好保护每个股东的权益与应该承担的责任。

股东可以担任财务负责人，监事不可以担任财务负责人。 财务负责人属于高级管理人员，为了保证监事的独立性，故参与公司经营管理的高管、董事均不得担任监事。

经营范围

这个就是看着你要做的关键业务流程是什么內容，选择经营范围只要在系统里选填就好了。经营范围不能填的太多，不然有可能提升公司税负；但也不能填的偏少，以防限制公司业务拓展。很多企业为了节省麻烦，把能想到的经营范围都写上去了。但是没有注意到，有些经营范围是不能享受税收优惠的。

注册地址

广州不可以用住宅地址注册公司，只能使用商用地址注册，自己无法提供地址但同时又不想租赁昂贵的写字楼办公室的话，可以使用挂靠地址注册。对于刚刚创业的小伙伴来说为了节省成本可以挂靠地址先进行注册。 有挂靠地址，在工商局有备案，不用担心地址异常，可接收信函，可供银行开户。

如果你没有注册地址可以使用挂靠地址注册，像视频类、电商类、咨询服务类、计算机类等不用实际办公地址的行业，完全可以使用挂靠地址注册公司。这种地址既可以解决创业路上的资金压力大问题，也可以解决目前无地址注册公司的难题！

纳税类型

注册公司是选择一般纳税人还是小规模纳税人？主要还得根据公司规模、行业来考虑。

  1. 看规模：如果公司投资规模大，年销售收入很快就会超过500万元，建议直接认定为一般纳税人。如果估计公司月销售额在10万元以下，建议选择小规模纳税人，从2019年1月1日起，将享受增值税免征政策。
  2. 看买方：如果你的购买方预计未来主要是大客户，很可能他们不会接受3%征收率的增值税专用发票，这样建议直接认定一般纳税人，否则选择小规模纳税人。
  3. 看抵扣：如果你的公司成本费用构成取得增值税专用发票占比高，进项税额抵扣充分，通过测算估计增值税税负低于3%，建议选择认定一般纳税人，否则选择小规模纳税人。
  4. 看行业：如果是13%税率的行业，且为轻资产行业。这类行业一般增值税税负较高，建议选择小规模纳税人，否则选择一般纳税人。
  5. 看优惠：看你所在行业能否销售增值税优惠政策。比如软件企业增值税超税负返还等增值税优惠政策，如果能享受，建议选择一般纳税人，否则选择小规模纳税人。
