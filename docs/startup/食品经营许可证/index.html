食品经营许可证

什么情况下需要办理《食品经营许可证》？

根据《食品安全法》规定，只要从事食品销售或餐饮服务活动（无论线上还是线下），均需办理该许可证。具体场景包括：

  1. 实体经营：超市、便利店、餐饮店、食品批发市场等；
  2. 线上销售：淘宝/拼多多等电商平台、微信小程序、独立网站、外卖平台；
  3. 其他形式：自动售货机、食品展销会、临时摊点等。

例外情况（无需办理）：

  • 销售食用农产品（如未加工的蔬菜、水果、生鲜肉禽）；
  • 仅销售预包装食品（需改为备案制，见后文说明）。

办理条件

1. 主体资格要求

  • 已取得《营业执照》，且经营范围包含“食品经营”相关类目。

2. 经营场所要求

  • 实体店：场地布局合理，与污染源保持安全距离（如远离垃圾场）；
  • 线上经营：需有实体仓储地址，并提供相关证明（如租赁合同）；
  • 设施设备：根据经营类目配置必要设备（如餐饮需消毒柜、冷藏需冰箱）。

3. 食品安全管理制度

  • 配备食品安全管理人员；
  • 从业人员需持有健康证明（可在社区医院办理）；
  • 建立进货查验、卫生管理等制度。

4. 特殊类目附加条件

  • 预包装食品：2021年起改为备案制（无需许可证，但需提交《仅销售预包装食品备案信息表》）；
  • 保健食品/婴幼儿奶粉：需专区专柜销售，并明确标注；
  • 冷链食品：需配备冷藏冷冻设备并定期测温。

办理流程（以线上+线下结合为例）

Step 1：准备材料

  • 营业执照复印件；
  • 法定代表人身份证；
  • 经营场所平面图、设备清单、卫生设施说明；
  • 食品安全管理制度文件；
  • 从业人员健康证明（至少1人）。

Step 2：提交申请

  • 线上：登录当地“政务服务网”（如浙江“浙里办”、广东“粤省事”），搜索“食品经营许可证”按指引提交；
  • 线下：到经营场所所在地的市场监督管理局窗口提交材料。

Step 3：现场核查

  • 监管部门收到申请后，5-10个工作日内派员到经营场所核查卫生条件、设备设施等；
  • 重点检查项： ✅ 是否与污染源保持距离； ✅ 冷藏/消毒设备是否齐全； ✅ 是否分区摆放食品（如生熟分离）。

Step 4：审批发证

  • 核查通过后，5-10个工作日内发放《食品经营许可证》（电子版与纸质版同步生效）。

有效期与费用

  • 有效期：5年（到期前30天需申请延续）；
  • 办理费用：
    • 官方不收取费用（国家已取消行政收费）；
    • 若委托代办公司，服务费约500-2000元（根据地区和服务内容）。

注意事项

  1. 备案制 vs 许可证：
    • 仅销售预包装食品（如袋装零食、瓶装饮料）只需备案，无需办证；
    • 若同时销售其他食品（如散装干货、自制糕点），仍需办理许可证。
  2. 合规风险：
    • 无证经营最高罚款5万-10万元；
    • 许可证信息变更（如地址、负责人）需在30日内申请变更。
  3. 线上平台要求：
    • 淘宝/京东等平台会强制审核许可证，无证将下架商品。

建议操作

  1. 先备案再升级：若初期仅销售预包装食品，直接备案节省时间；后期扩展业务（如增加自制食品）再补办许可证。
  2. 咨询当地部门：不同地区对“经营场所”定义可能不同（如住宅能否作为仓储），建议提前电话咨询市场监管局。
  3. 健康证明代办：可联系社区卫生服务中心批量办理员工健康证，费用约50-100元/人。

如果仍有疑问，可拨打全国市场监管热线 12315 或登录属地政务服务网查询具体细则。

FAQs

什么是预包装食品？

根据《食品安全法》和《预包装食品标签通则》（GB 7718）的规定：

预包装食品指 预先定量包装 或 制作在包装材料和容器中，并且在一定量限范围内具有统一质量或体积标识的食品。

简单来说，这类食品在 生产环节就已密封包装，消费者购买时无需商家二次加工或分装。

核心特征：

  1. 预先包装：出厂时已密封（如袋装、罐装、盒装），直接面向消费者销售。
  2. 定量标识：包装上明确标注了 净含量（如“净含量500g”）。
  3. 统一规格：同一批次产品规格一致（如每瓶饮料均为330ml）。

常见例子：

  • 零食类：袋装薯片、盒装饼干、罐装坚果；
  • 饮料类：瓶装矿泉水、罐装可乐、盒装牛奶；
  • 粮油类：袋装大米、瓶装食用油、预包装调味料（如袋装盐、酱油）；
  • 速食类：方便面、自热火锅、真空包装卤蛋。

表：与“散装食品”的区别

维度预包装食品散装食品
包装形式出厂时已密封，不可拆零销售无固定包装或大包装，需商家称重分装销售
标签要求需符合 GB 7718（强制标生产日期、成分表等）仅需标注品名、保质期、产地等基本信息
销售场景超市货架、电商平台菜市场干货摊、超市散装区
监管要求仅销售预包装食品可备案，无需许可证需办理《食品经营许可证》

注意事项：

  1. 标签合规性： 预包装食品必须完整标注： ✅ 生产日期、保质期； ✅ 配料表、营养成分表； ✅ 生产商名称、地址、SC许可证编号； ✅ 储存条件、食用方法（如需要）。 （标签不合规可能被平台下架或面临处罚，标签审核可委托专业机构或咨询市场监管局，避免因标签问题被处罚。）
  2. 特殊品类：
    • 进口预包装食品：需加贴中文标签，并提供《入境货物检验检疫证明》；
    • 保健食品：需标注“蓝帽子”标识及批准文号（如“国食健注G2023XXXX”）。
  3. 线上销售要求： 电商平台（如淘宝、京东）通常要求上传备案凭证或食品经营许可证，并核对标签信息。

备案制 vs 许可证？

  • 仅销售预包装食品： 2021年起，国家将预包装食品销售改为 备案制（无需办理许可证），仅需向市场监管部门提交《仅销售预包装食品经营者备案信息表》。 备案内容：经营者名称、地址、销售食品类别等基本信息。
  • 若同时销售其他食品： 例如在卖预包装饮料的同时销售现制奶茶，则需办理《食品经营许可证》。

备案流程：登录当地政务网（如上海“一网通办”）在线提交备案，通常1-3个工作日完成；

什么是农副产品？

农副产品 指 农业活动中生产的各类初级产品及其简单加工品，涵盖 种植业、畜牧业、渔业、林业 等领域的产出。 根据《农产品质量安全法》，农副产品分为两类：

  1. 未经加工的初级农产品：如新鲜蔬菜、水果、活鱼、原木、生鲜禽蛋等；
  2. 初加工农产品：仅经过简单物理处理（如清洗、分拣、晾晒、包装），未改变产品基本自然性状的农产品，例如：
    • 分装的大米、干香菇；
    • 晾晒的红枣、腌制但未添加防腐剂的咸菜；
    • 冷冻的肉类、切割分装的鲜切花。

核心特征：

  • 来源：直接依赖农业生产；
  • 加工限度：未改变产品化学性质或核心物理形态；
  • 用途：多作为食品原料或初级消费品。

什么是自产农副产品？

自产农副产品 特指 由农业生产者（个人或企业）在其自有或租赁的土地、养殖场等场所中直接生产并销售的农副产品。 例如：

  • 农户自家菜地种植的蔬菜；
  • 农场养殖的活禽、禽蛋；
  • 渔民捕捞后直接售卖的鲜鱼。

政策优惠：

国家对自产农副产品销售有税收和监管简化政策：

  1. 免税政策：自产自销的初级农产品免征增值税（需开具免税发票）；
  2. 无需许可证：销售自产的初级农产品（未加工）通常无需办理《食品经营许可证》；
  3. 简化备案：部分初加工产品（如分装干货）可能仅需备案。

表：农副产品和自产农副产品两者的核心区别

维度农副产品自产农副产品
生产主体任何主体（包括收购转售的经销商）仅限农业生产者（如农户、合作社、农场）
加工深度可能包含初加工（如分装、腌制）通常为初级产品或简单初加工
政策待遇经销商需纳税，初加工品可能需备案/许可自产自销免税，初级产品免许可证
销售限制无特殊限制需证明“自产”（如土地承包合同、生产记录）

简单总结：

  • 农副产品 是广义的农业产出概念，涵盖所有生产主体和加工层级；
  • 自产农副产品 是农副产品的子集，强调“自产自销”，享受政策便利；
  • 核心区别 在于生产主体身份和监管要求，需通过证明材料和加工深度界定。

如何界定“自产”？

需提供以下证明材料（根据各地政策可能不同）：

  • 土地/养殖场承包合同；
  • 生产记录（如种植日志、养殖台账）；
  • 村委会或农业部门出具的产地证明。

注意：若对农产品进行 深加工（如烘焙坚果、制作果酱），则视为食品生产行为，需办理《食品生产许可证》。

能否线上销售？

  • 初级自产农产品（如新鲜蔬菜、水果）：可在电商平台（如拼多多、微信小店）直接销售，无需许可证，但需提供产地证明；
  • 初加工自产农产品（如干木耳、蜂蜜）：若仅经过简单处理，一般仍按农产品管理；若添加防腐剂或深加工（如制成蜜饯），则需办理食品生产许可证。

标签要求？

  • 初级农产品：需标注品名、产地、生产者/销售者信息；
  • 初加工农产品：若分装销售，需标注分装日期、分装企业信息；
  • 深加工农产品（如瓶装辣椒酱）：需按预包装食品要求标注完整信息（成分表、SC编号等）。

经销商需要办理许可证吗？

经销商不需要申请《食品经营许可证》，仅需办理《仅销售预包装食品备案》。

即使供应商已持有《食品生产许可证》或《食品经营许可证》，作为经销商，你仍需完成备案。这是国家针对预包装食品销售的最新监管要求（2021年修订版《食品安全法》）。

法规依据：

  1. 《食品安全法》第三十五条：仅销售预包装食品的经营者，应当报所在地县级以上市场监管部门备案。
  2. 《市场监管总局关于仅销售预包装食品备案有关事项的公告》：明确取消“仅销售预包装食品”的许可审批，改为备案管理。

备案条件：

  • 持有《营业执照》，且经营范围包含“食品经营（仅销售预包装食品）”；
  • 销售的商品为 未经拆封、未二次加工 的预包装食品（如直接从供应商进货的袋装零食、瓶装饮料等）。

备案材料：

  • 营业执照复印件；
  • 经营者身份证（个体工商户）或法定代表人身份证（企业）；
  • 经营场所证明（线上销售可提供仓储地址租赁合同或自有产权证明）；
  • 供应商资质文件（非备案必需，但建议留存备查）： ✅ 供应商的《食品生产许可证》或《食品经营许可证》； ✅ 产品出厂检验合格证明。

备案流程：

  • 线上办理：登录属地政务服务平台（如浙江“浙里办”、广东“粤省事”），搜索“仅销售预包装食品备案”，在线填写信息并提交；
  • 线下办理：前往经营场所所在地的市场监管局窗口提交纸质材料；
  • 办理时效：1-3个工作日完成备案，获取《仅销售预包装食品备案信息采集表》（无纸质证书，备案信息可在政务平台查询）。

备案后注意事项：

  • 平台入驻：淘宝、微信小店等平台会要求上传备案凭证（截图政务网备案信息即可）；
  • 信息变更：若经营地址、负责人等信息变动，需在15日内更新备案；
  • 无需年审：备案长期有效，终止经营后需注销备案。

常见误区澄清：

  • 误区1：供应商有证，我不需要备案

    正解：备案针对的是 你的销售行为，与供应商是否持证无关。即使供应商合法，你作为独立经营者仍需备案。

  • 误区2：线上销售无需经营场所证明

    正解：线上店铺虽无实体门店，但需提供 食品仓储地址（如租用仓库或家庭住址），该地址需符合食品安全存放条件（如防潮、防鼠）。

  • 误区3：备案后可以销售任何预包装食品

    正解：若销售 特殊类别（如保健食品、婴幼儿配方奶粉），需在备案时勾选对应类目，并遵守额外规定（如专区销售、标注警示语）。

建议操作流程：

  1. 第一步：办理营业执照
    • 经营范围需包含“食品经营（仅销售预包装食品）”。
  2. 第二步：提交备案
    • 在线完成备案，保存备案编号。
  3. 第三步：审核供应商
    • 要求供应商提供《食品生产许可证》及产品合格证明。
  4. 第四步：平台入驻
    • 上传备案信息至淘宝/微信小店等平台，通过资质审核后上架商品。
