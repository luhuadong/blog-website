博客阅读指南

大家好，我是阿基米东，「成为自由人」是我的个人网络日志频道，这里会记录我的一些日常、思考，以及想分享的知识内容。在 About 页面有关于我的个人简介，可以快速了解我。

这些年，虽然感觉自己懂很多道理，但是却依然过不好每一天的生活，包括面对的人际关系、感情、工作、创业，所以我在《人生之书·我的原则》中记录了一些为人处世的原则。很多都是因为犯过错才反省出来的，希望以后出现类似情况可以从容面对，不会在同一个地方再次摔倒。

从 2016 年开始，我每年都会写一篇总结，记录过去一年发生的事情和个人成长，如果你感兴趣，可以看一看。

  • 我的2023——现实的理想主义者
  • 我的2022——三十不惑
  • 我的2021——为自己的选择放手一搏
  • 我的2020——一颗红心，两手准备
  • 我的2019——努力成为我认为我可以成为的人
  • 我的2018——上懂机器学习，下懂微信建群
  • 我的2017——作为一名程序员，我重新思考了我生活的目标
  • 我的2016——我和嵌入式的那些事

同时，不定时日记 栏目也会一直更新，不过更多是记录我自己的一些生活，你也不一定感兴趣吧。

更多的，我希望这个博客能够分享一些对大家有用的知识。其中一个重点，就是如何获得快乐、幸福和自由。就我自己而言，我会经常翻看《新幸福的十大标准》这篇文章，让自己在失落迷茫的时候保持前进的动力。

我没什么兴趣爱好，目前来说主要就是「读书」和「运动」两大板块，所以 一起读书呀 和 一起运动呀 两个栏目会分享一些读书笔记和体育运动的技巧。

  • 图书方面，我目前比较阅读较多的除了编程类工具书，基本上集中在社会学、心理学、婚恋、环保、心灵成长等领域。
  • 运动方面，由于早期短暂的运动员生涯经历，所以常见的运动都会一点，除了跑步、游泳、健身，比较擅长的球类运动包括篮球、橄榄球、羽毛球，另外足球、网球和乒乓球也略懂一二。不过，由于年纪大了，也没有继续往运动员方向发展，所以现在更感兴趣的是普通人如何通过体育锻炼提高身体素质和意志力，以及如何通过体育运动促进亲子关系。

鉴于目前依然是一名打工人，自认有较为丰富的 IT 领域职场经历，曾在国企、私企、创业公司、开源项目等环境工作过，也曾有过两段小打小闹的创业经历，体验过从几个人到几千人的工作环境。因此，在 了不起的打工人 栏目里会分享一些关于程序员职业规划、IT 项目管理等内容。

这里还有一个 一起学编程 的栏目，最早是因为 2019 年寒假回家想教表妹学 Python 编程写的《教孩子学编程》专栏，一共10天。后面由于创建了「人人都懂物联网」网站，与编程技术相关的内容都在上面更新了，所以这个博客估计暂时不会有太多编程教程。

除了编程，我对环保也很感兴趣，甚至出现了“环保强迫症”，一起零废弃 栏目记录了一些和环保、零废弃、可持续相关的内容。不过，由于后面也创建了一个独立的网站「零废弃部落」，所以环保相关的内容都会优先在那里更新，感兴趣的朋友可以关注一下。

有时候我也很懊恼，我在 2017 年的时候同时得到了两个梦想 —— 物联网 & 环保。一度让我的生活陷入了混乱，世界观也经历了重建，因为两个圈子有太多不一样了，对我来说像是两种割裂的生活。直到 2021 年前后，这种痛苦的现状才有所缓解，我也在两个领域中找到了一些连接点，作为一名创客和环保主义者，我越来越相信未来有一天，两者会融合起来。《GetIoT 信息列表》和《ZeroTogether 信息列表》记录了一些我经常浏览的网址，可以说是我早期学习如何成为一名创客和环保主义者的资讯来源。

要想成为自由人，精神自由、财务自由、时间自由缺一不可。精神自由可以通过持续地学习、锻炼、思考等方式获得；财务自由则需要提高我们的财商，好好赚钱 栏目会分享一些这方面的内容；而时间自由，除了实现财务自由后不用上班腾出来的大把时间，更重要的是要学会好好管理时间，人生管理术 栏目会分享一些时间管理方面的内容。

此外，本博客还会分享一些 素食菜谱、商业案例、医疗健康、城市地理、实用地图、纪录片、演讲 以及各种 奇奇怪怪定律 等内容，它们都是我个人感兴趣或平常用得上的东西。如果你也刚好需要，那就挺好的。更多的栏目，可以在 全部分类 中查看。
