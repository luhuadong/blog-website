关于 😎

阿基米东（英文名 Rudy）是一位善良、可爱的同学，他曾经是一个有梦想的小镇做题家，当过中考状元，念过大学，摆过地摊，做过程序员，创过业，被社会毒打十年后，现在已经变成了一个有梦想的30+大叔。

他年轻的时候曾沉迷于体育运动，学生时代主要练习中长跑🏃‍♂️，同时也练习过篮球🏀、排球🏐、橄榄球🏈等运动项目，大学时期曾担任橄榄球队队长，但遗憾并没有取得好成绩。在认清理想和现实的差距后，他放弃了走职业运动员路线，转身投入到计算机编程中，并在大学毕业后顺利成为了一名程序员🧑‍💻。现在，他最喜欢的运动是篮球🏀、游泳🏊‍♂️和爬山🧗‍♀️，也是雪狼队（三人篮球）成员之一。

他并非计算机科班出身，而是电气工程及其自动化专业毕业的，因此在学习编程这件事情上吃了很多苦头。幸好他在刚开始学习编程的时候就接触到 Linux 系统，被纯粹的、自由的开源文化深深吸引，这给了他足够的力量坚持下去，也是从那时候起，他开始在 CSDN 写博客，分享自己的学习笔记。后来，他在工作之余，参与了一些开源项目，也给 Linux、RT-Thread、LVGL、openEuler 等开源项目贡献过代码（这里记录了我常关注和参与的项目），在这个过程中感受到了极大的编程乐趣，也提升了个人技术水平。在 2020 年，他还获得了 “RT-Thread 开源社区杰出贡献奖” 🏆，也跟随 RT-Thread 核心团队登上过一期 Gitee 封面人物，这让他非常开心！现在，他在一家企业担任软件架构设计师的角色，业余时间在开发和维护他自己创建的 OpenEmber 开源项目。

他的涉猎非常广，虽然早期通常作为嵌入式软件工程师，但他并没有因此给自己设限，凡是他感兴趣的知识，他都想学。所以后来自学了 Web 和 Python 全栈开发，开始自诩为一名全栈开发者。现在，他懂一点硬件、系统、前端、后端、UI、产品，并以成为一名杰出的 Indie Hacker 为职业目标，希望能以一种更自由、灵活的方式做一些有价值的产品，他实在太爱自由了！

如果不以工作岗位来介绍自己，他喜欢称自己为 Maker（创客）或 Hacker（极客）。他也是 柴火创客 的年度认证会员，并希望在业余时间能组织或参与一些技术沙龙和工作坊，比如他很喜欢的 freeCodeCamp 社区活动、RT-Thread 沙龙聚会、openEuler Meetup 等等。

除此之外，他还有一个很重要的标签，那就是 “环保主义者”🍀。现实生活中，他有一点环保强迫症，他不用塑料袋，不用一次性筷子，不点外卖，不喝奶茶，甚至还不怎么吃肉。最近，他还开始研究怎么做素食和营养搭配，以此达到成为“纯净素”的目的。他在 2018 年的时候曾在一家环保机构（NGO）工作过，因此对环保、公益领域有一定的了解，并且希望能够用 IT 技术赋能社会可持续发展、逆转全球变暖。因此，他曾做过一个开放的空气监测数据平台「FCTC-Air」（目前已关停），创建了一个关注自然、低碳、环保、可持续生活的网站「零废弃部落」，以及一个面向大众的免费、平等的物联网（IoT）技术教育网站「人人都懂物联网」。

另外，他还在密谋着撰写书稿、录制播客和视频，希望在不久的将来能够大大方方地给自己贴上 “图书作者”、“主播”、“Up主” 等标签，同时获得更大的影响力去实现心中梦想。

反正，他就是一个爱学习、爱折腾的人，他在不断完善自己，也在努力创造一份对社会有益的事业。当然，他也希望在实现人生价值的同时，也有自由的时间陪伴重要的人，一起走过人生美好的旅程，一起看山⛰️、看云🌥️、看花🌺、看海🌊 ......

最后，欢迎订阅他的微信公众号「成为自由人」，虽然不会定期更新，但还是欢迎您~
