WordPress 的 WiKi 知识库主题

WordPress 适合做内容管理网站，为了做出像 Wikipedia 那样的风格来展示网页内容，就需要主题的配合。本文列举一些也许适合你的 WordPress Wiki 类型主题，帮助你搭建自己的知识库！

KnowAll

KnowAll 是一个灵活的 WordPress 知识库和 wiki 主题，具有内置功能，可帮助访问者找到他们正在寻找的确切信息。你可以使用它来创建一个完整的知识库。

KnowAll 是一个高端主题。它在单个网站上的使用价格为149美元，并提供一年的支持和更新，但如果你想访问搜索分析和文章反馈功能，你就需要购买199美元的版本。

WikiPress

WikiPress 是一个专用的 WordPress wiki 主题，具有干净、响应式的布局。该主题的左侧有一个侧边栏，它会自动在按类别组织的折叠手风琴部分中显示你的所有内容。另外，顶部还有一个实时搜索框。

这个主题最有趣的地方在于，以真正的 wiki 方式，它允许人们使用常规的 WordPress 编辑器从前端添加、编辑和管理内容。

WikiPress 是一个优质的 WordPress 主题。一年的支持和更新费用为 99 美元。第一年后，你可以以每年 49.50 美元的折扣价续订以继续获得支持和更新。

KnowHow

KnowHow 是另一个灵活的 WordPress 知识库和 wiki 主题。它可以帮助你添加不同主题的文章并将它们组织成易于搜索的类别。访问者还可以利用即时搜索建议更快地找到内容。

KnowHow 是一个高级主题，售价 59 美元。它是通过 ThemeForest 出售的，这意味着你的购买可以让你访问单个站点的终身更新和六个月的支持。

  • 示例 Demo

Docly

顾名思义，Docly 是一个致力于帮助您创建文档和 wiki 网站的主题。你可以将它用于任何类型的文档、wiki、知识库等。

一个独特的功能是它带有一个集成的支持论坛。因此，如果你想创建一个 wiki 并允许公开讨论，那么这个可能是一个不错的选择。

Docly 是一个高级主题。ThemeForest 售价 59 美元，可让你访问单个站点的终身更新和六个月的支持。

Manual

与其他 wiki 主题一样，Manual 是另一种通用选项，可用于 wiki 站点、知识库，并且通常只显示任何类型的文档。

与 Docly 一样，如果你想将讨论论坛添加到你的 wiki，它还与流行的 bbPress 插件集成。

Manual 是 ThemeForest 提供的高级主题。与其他 ThemeForest 主题一样，它的价格为 59 美元，并为您提供单个站点的终身更新和六个月的支持。

Heroic Knowledge Base

Heroic Knowledge Base 为你的站点添加了创建功能齐全的知识库所需的所有功能。通过选择此插件，你可以使用任何你喜欢的 WordPress 主题来构建你的在线支持资源。

Docy

Docy 是一个功能强大、响应迅速且高性能的在线文档 WordPress 主题，可以帮助你快速开发下一个产品的文档。你可以有效地创建文档以记录 API、框架、插件、软件、模板、任何类型的产品。它包含许多元素，使你的文档具有交互性和效率。这个主题是完全响应和移动友好的。

Wikb

Wikb 旨在帮助你使用 WordPress 启动知识库和帮助台网站。

如果你的客户或用户在使用你的产品或服务时需要支持，那么 Wikb 主题就是为你而构建的。Wikb 主题也适用于设置在线空间，你可以在其中发布内容以帮助你的员工、社区或任何其他人群。

Wikb 模板的设计非常适合此类产品。整体风格严肃而热情，让你的访问者清楚地知道他们可以信任你发布的信息，同时仍然对你的网站感到宾至如归。由于所有模板都具有非常现代的设计，因此你使用 Wikb 创建的任何网站都应该对你的品牌产生积极影响。

weDocs

这个免费的WordPress wiki插件提供了一个简单的解决方案来创建和管理知识库文章和文档页面。与此列表中的其他WordPress插件类似，用户可以使用拖放功能对文章进行分类和重新排列。

此外，它还具有用于文档搜索和目录的专用小部件。

weDocs的界面让用户可以在一个屏幕上管理所有的wiki文档。这个免费插件不提供模板，而是与最流行的WordPress主题无缝集成。

此外，它从外部站点导入文档，以便用户可以利用现有文档创建知识库。

此外，weDocs还为知识库提供了内置的电子邮件回复系统。它允许网站访问者提供有关内容的反馈，让支持团队和其他管理员深入了解如何改进他们的WordPress网站。

Echo Knowledge Base

该插件提供26种预制和响应式设计，适用于任何WordPress主题。它具有粘性目录，这意味着它会在读者滚动时保留在文章的一侧。

如果您想在不浏览不同设置的情况下使用此WordPress插件，设置向导将帮助您立即创建知识库。

创建一些文章和类别后，您可以按字母顺序或时间顺序对它们进行排序，添加标签、面包屑、后退按钮和评论。此外，Echo包括一个自动生成的目录。

此外，该插件针对最佳SEO结果进行了编码，这对于所有专注于通过其知识库内容提高在线知名度的网站所有者非常有帮助。

这个WordPress wiki插件带有三个布局选项——基本、选项卡和以类别为中心。它还有一个可视化布局编辑器，允许用户修改颜色、标签、字体和样式，并预览所有更改。

Echo知识库的 Pro 版高级功能包括一个访问管理器，用于控制哪些用户可以创建或阅读文章、文章评级和站点访问者的反馈等。计划起价为46美元/年。
