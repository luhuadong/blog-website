摄影术语入门（摄影名词解释）

拍摄模式（Camera Modes）

相机通常预设多种拍摄模式用于控制光圈、快门和 ISO 感光度，例如全自动、光圈优先、快门优先、程序模式和手动模式等。建议新手学习并掌握光圈优先、快门优先和手动模式，它们可以覆盖大部分的拍摄场景。

  • 光圈优先（Aperture Priority Mode）：由你来控制相机的光圈和感光度，而快门速度会由相机根据测光表来决定。
  • 快门优先（Shutter Priority Mode）：由你来控制快门速度和感光度，而光圈则由相机根据测光表来决定。
  • 程序模式（Program Mode）：由你来控制感光度，而光圈和快门则由相机决定。
  • 手动模式（Manual Mode）：你可以完全控制相机的光圈、快门和感光度，相片的曝光和动态完全由你决定，这是最自由的拍摄模式。

曝光三要素

曝光三要素：光圈、快门、ISO。

光圈（Aperture）

光圈位于镜头内，由光圈叶片控制大小。通常由 f 数值表示，例如 f2.8 / f4 / f8 等。光圈大小会对进入相机的光线有所影响。

  • 光圈的大小影响了曝光和景深。
  • 光圈越大，进光量越多，图片越亮。这就是光圈对曝光的影响。
  • 大光圈可以创造很浅的景深，也就是前景和背景的模糊程度，在人像摄影，或者花卉，静物拍摄时，常常用到。而在风光摄影中，比较常用到小光圈。

快门（Shutter）

快门速度是指相机内的快门打开的时间。快门越快，越可以凝固高速移动的物件；而慢快门则可以记录物体移动的时间。

  • 快门用来控制进光的时间，可以凝固动作或者创造模糊效果。
  • 利用不同的快门速度，我们可以把高速移动的物体凝结住，例如飞行的鸟类，飞溅的水花等等，也可以拍出丝绢流水的效果，利用慢快门，我们甚至还可以拍出星轨。

下面是常用的快门速度，可供大家参考使用。

快门速度通常运用场景
1/4000冻结极快运动。
1/2000冻结飞行中的鸟类。
1/1000冻结摩托车、汽车等高速行驶的车辆。
1/500冻结骑行中的自行车、跑步者，或者运动员。
1/250冻结慢速移动的人或者动物。
1/125摇拍摩托车、汽车等高速行驶的汽车。
1/60摇拍近处的山地自行车。
1/30摇拍快速移动的自行车，并且有一定距离。
1/15摇拍跑步者，移动的孩子或者动物。
1/8近处快速流动的溪水拍出模糊的效果。
1/4模糊拍摄人物的步行。
1/2模糊低速流淌的水。
1秒或更慢丝绢流水的效果。

ISO（感光度）

ISO 也就是感光度，是相机对光线的敏感度。感光度越高，要正常曝光所需的光线就越少，但噪点也会相对增加，清晰度下降。所以我们应该使用合适的感光值，不要用非必要的高感光度。

ISO 越高，照片越亮。但是过高的 ISO 会产生噪点，让图片画面质量变差。如果你的相机不够好，或者你的照片需要放大来看，那么建议你还是用低 ISO 来拍摄。因为提升亮度并不一定要提高 ISO，可以用大光圈，慢快门，调整 EV 等等。

在摄影中，ISO 等同于「感光度」的意思，它代表了底片（感光元件）对于光线的敏感程度。ISO 的英文全称是 International Standardization Organization，即国际标准化组织。因为是 ISO 组织把感光度做了量化的规定，所以大家也就把 ISO 叫做感光度了。

不过，最早的 ISO 感光度标准是针对胶片相机的，在数码相机时代，则把感光元件对于光线的灵敏度等效为传统胶片的感光度值。因此，数码相机 ISO 的严谨的叫法应该是「相对感光度」或者「相当感光度」。

曝光（Exposure）

曝光分为曝光不足，曝光过度和曝光正确三种情况。

数码相机中有一个标有 -2~+2 的曝光计，在相机自动测光的基础上，“-”表示曝光不足，“+”则表示曝光过度，0表示适当曝光。在通常情况下，以相机自动测光后得到的数值为基准，来判断是否曝光不足，适当曝光及曝光过度。

白平衡（White Balance）

白平衡就是相机的 AWB，用来调整相机捕捉的色彩。

一张照片的组成部分除了曝光（光圈、快门、感光度），颜色也是非常重要的一环。而要控制颜色，就需要设定相机的白平衡。现代数码相机一般有很多预设的白平衡模式，例如日光、阴影、阴天、白炽灯、荧光灯、日光、闪光灯等。当然，你也可以自定义白平衡的值。

简单来说，白平衡会影响照片的色温，通过调节白平衡，同一张照片我们可以产生完全不同的视觉效果。不过，白平衡虽然重要，但是我们完全可以在后期进行调整，在转瞬即逝的拍摄瞬间，建议大家不要把注意力放到白平衡上。

画幅

画幅，在胶片时代指的是胶片的尺寸规格，现在指的就是相机传感器（感光元件，CCD/CMOS）的尺寸。 全画幅相机的感光元件相当于一个胶片的大小（36×24 mm），而半画幅只有胶片的一半大小。相对来说，全画幅相机的画质更好、细节也更丰富。

全画幅（Full-frame）

全画幅相机是针对传统 135 胶卷的尺寸来说的。以前大部分的数码单反相机 CCD 尺寸都比 135 胶卷的尺寸小，而全画幅数码单反的 CCD（或 CMOS 的感光成像的元件）尺寸和 135 胶卷的尺寸相同，保持 36×24mm 规格为 135 格式的胶片大小。

半画幅（Half-frame）

半画幅也叫 APS-C 画幅（全称 Advanced Photo System type-C），其感光元件尺寸为 23.5×15.6mm 左右，全画幅的感光元件尺寸大约是半画幅相机的2.3倍。
