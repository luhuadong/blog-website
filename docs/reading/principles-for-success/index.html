《成功的原则：超迷你探险系列》动画

概述

瑞·达利欧（Ray Dalio）26 岁白手起家创办桥水基金，把桥水打造成全球最大的避险基金公司。瑞·达利欧将生活、管理、商业和投资系统化为一系列「原则」，并认为是原则造就了他的事业成功，对工作和人际关系的价值更是弥足珍贵。2017年他基于自己的生活和工作经历而撰写的 600 页《原则》巨作，自上市以来，全球已有数百万人阅读和分享，包括如何设定目标、从错误中学习以及与他人合作以产生卓越成果。

瑞·达利欧认为，是「原则」成就了他和桥水的空前成功，处在当下 70 岁的第三人生阶段，他更希望能够透过自己的分享让更多人实现目标，达成使命。为了让更年轻的读者亲近与理解，瑞·达利欧截取了《原则》的精华拍摄为「成功的原则」动画影片，进而改编成一本易读有趣的绘本《成功的原则》，献给所有年龄层的读者。

视频

《成功的原则》动画共 8 集，约 30 分钟。第一集：冒险的召唤；第二集：拥抱现实，应对问题；第三集：五步原则；第四集：深渊；第五集：一切都是机器；第六集：两大障碍；第七集：保持思想开放；第八集：好好奋斗。

笔记

1、你必须自己思考真相是什么？

  • 犯错误，反思错误，总结出原因，改正错误
  • 跳出舒适圈，有勇气为此付出行动
  • 真相是取得良好结果的根本基础
  • 梦想 + 现实 + 决心 = 成功的生活

2、拥抱现实，应对现实

  • 克服（错误、问题、弱点）
  • 痛苦 + 反思 = 进步

3、五步流程

  1. 制定目标（根据自身确定你的目标）
  2. 发现问题（遇到问题，不要容忍）
  3. 诊断问题（反思为什么会出现这种问题？找出根本原因）
  4. 规划方案（找到处理的办法，设计解决方案）
  5. 执行方案（敦促自己去解决）

成功的人生包括一次又一次完成这五个步骤，不断循环，使你的个人进化迈向成功的更高阶关卡。

4、深渊

  • 客观面对失败
  • 平衡风险与回报

5、一切都是机器

  • 类似情境的重现
  • 每件事都因果关系的
  • 眼界放宽，从历史中找到规律

6、你的两大障碍

  • 自我意识障碍（不要太自以为是，敢于接受缺点）
  • 思维盲点障碍（接受不了自己认知以外的。每个人对事物、环境的理解不同，观点也不同。）

7、做到头脑极致开放

  • 不要只看对错，而是寻找真相
  • 不要看重结论，要学习不同视角，看问题的思路
  • 对不同观点的包容，才能发现自己看不见的问题，不是盲目听信

8、奋力拼搏

总结

  • 要成功就要勇于冒险

    知道如何在风险与报酬之间取得适当的平衡，是创造美满人生不可或缺的元素。专注于实现自己的梦想，勇于承担并且去理解影响你的现实问题，找出妥善处理之道，进而下定决心去做，你就学会了足以造就成功人生的原则。

  • 与聪明人当队友

    穿越人生丛林的最好方式，是与一群看事情角度与你不同的有识之士结伴同行。你可以跟别人合作，取长补短。

  • 练习你的人生旅程

    这个练习是为了帮助你把自己的人生、你所关心的人的人生纳入全盘考虑，并对未来做好计划。在这个练习中，可以看看典型的人生旅程，省思自己的生活，以及自己可能会遇到的问题。

希望这本书、这部动画能引导你思考人生、得到成长、获得成功，实现自己想要的生活。
